# Xuqi-Daoqi Blueprint

The complete engineering blueprint connecting Huaxia tradition (虚器 / 道器) with modern AI system design. The package includes the original hand-drawn blueprint, the engineering translation, annotated version, audit report, and mind map.

## Files

- `00_README.pdf` — Package guide
- `01_Blueprint_Original_20260628.pdf` — Original blueprint
- `01_Blueprint_Original_Complete_20260628.pdf` — Original complete version
- `01_Blueprint_Original_Redrawn_20260628.pdf` — Redrawn original
- `02_Blueprint_Engineering_v2_20260628.pdf` — Engineering translation
- `03_Blueprint_Annotated_20260629.pdf` — Annotated version
- `Xuqi_Daoqi_Complete_20260628.pdf` — Complete compiled package
- `Appendix_Audit_Report_20260628.pdf` — Audit report
