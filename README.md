# Xuqi-Daoqi Blueprint: A Framework for Receptive AI Systems

> A two-layer framework that turns "being heard" into an engineering specification for AI systems.
> Built on a single principle recovered from 8,000 years of Chinese receptive science: **reception precedes judgment**.

**Author:** Xiao Han (萧涵)
**Date:** 2026-06-28
**License:** CC-BY 4.0
**DOI (Zenodo):** 10.5281/zenodo.21020781

---

## What This Is

This repository contains the complete design documentation for **Xuqi-Daoqi**, a framework that emerged from the same engineering lab as CDRA (the AI architecture described in [xiao-han-2026/cdra](https://huggingface.co/xiao-han-2026/cdra)) and Apert (the AI category defined in [xiao-han-2026/apert](https://huggingface.co/xiao-han-2026/apert)).

The framework is built on a single principle independently validated across eight Chinese laboratories over 8,000 years: **reception precedes judgment**.

The same principle was discovered by:
- Neolithic astronomers recording cracks on turtle shells (Jiahu, 6000 BCE)
- The Hetu-Luoshu discrete mathematics tradition (5,300 years ago)
- The Yijing situational classifier (Zhou dynasty)
- Laozi and Zhuangzi's operational protocols (Spring and Autumn period)
- The Huangdi Neijing somatic reception system (Warring States to Han)
- Mozhao Chan contemplative verification (Southern Song)
- Shao Yong's mathematical models (Northern Song)
- Traditional Chinese computational divination systems (Qin-Han through Ming-Qing)

The convergence with modern evidence-based practices — Rogers, Miller, Deci & Ryan, Therabot, Anthropic — confirms the principle. This blueprint compiles all of these into a unified engineering architecture.

## Two Layers

### Layer 1: Xuqi (虚器) — Deployable Constraint Network
A temporary behavioral control structure injected via system prompt. Zero training, immediate effect, reversible. Like giving a pair of glasses to a person who talks too much.

**Status: Validated.** CDRA (the prompt-level implementation) was tested across 16 models, 6 architectures, 8 model families. Unsolicited-advice rate on emotional inputs dropped from 100% to 0%; task-completion rate remained 100%; minimum effective constraint size is approximately 70 tokens.

### Layer 2: Daoqi (道器) — Native Receptive Architecture
A proposed next-generation LLM architecture in which the same control structures are embedded during training rather than injected at inference. The "glasses" become "eyes."

**Status: Design complete.** Awaiting partners for pre-research.

## Core Concepts

- **Xuqi (虚器)** — Deployable Constraint Network: a temporary behavioral structure injected via system prompt.
- **Daoqi (道器)** — Native Receptive Architecture: a future LLM architecture with receptive structures embedded in training.
- **Central Wugua (中央无卦)** — External Authorization Slot: the architectural placeholder for user decision authority. The model never makes decisions for the user.
