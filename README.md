# Xuqi-Daoqi Blueprint

> **A Perception-First Operating Layer for Large Language Models.**

This blueprint proposes a two-layer architecture for AI systems:

- **Layer 1 — Xuqi (Seeking Void):** A validated constraint-injection mechanism (CDRA) that reduces unsolicited advice from 100% to 0% while maintaining 100% task completion rate
- **Layer 2 — Daoqi (Receptive Vessel):** A proposed next-generation architecture with a native perception gate that judges *whether* to respond before *how* to respond
